Git Resources

En este sitio se guardaran los principales comandos de Git y se explicaran a detalle para que se usan
Git init : Se inicia Git para nuestro proyecto de desarrollo guardado en la carpeta, para esto tenemos que estar ubicados en la terminal del editor de codigo en la carpeta respectiva de nuestro proyecto
Git add . : Agrega todos los archivos que fueron modificados al repositorio temporal de Git
git commit -m "" : Agrega los archivos modificados al repositorio en los que se le hace una salva, podemos nombrar esta salva en las comillas despues de -m
